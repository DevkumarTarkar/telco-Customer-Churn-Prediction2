
# 📊 Telco Customer Churn Prediction using Machine Learning

A Machine Learning project that predicts whether a telecom customer is likely to churn using **Random Forest Classification** and performs **Customer Segmentation** using **K-Means Clustering**.

---

## 🚀 Project Overview

Customer churn is one of the biggest challenges faced by telecom companies. This project helps identify customers who are likely to leave and groups customers into meaningful segments for targeted retention strategies.

---

## ✨ Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Random Forest Classification
* Hyperparameter Tuning
* Customer Segmentation using K-Means
* Feature Importance Analysis
* Cross Validation
* Model Evaluation (Accuracy, Precision, Recall, F1-Score, ROC-AUC)

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📂 Dataset

**Dataset:** Telco Customer Churn Dataset

The dataset contains customer demographic information, services, billing details, contract information, and churn status.

---

## 📈 Machine Learning Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Train-Test Split (80:20)
5. Random Forest Model Training
6. Model Evaluation
7. Customer Segmentation (K-Means)

---

## 📊 Evaluation Metrics
